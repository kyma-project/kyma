# Custom Resources <!-- {docsify-ignore-all} -->

## APIGateway Custom Resource

The `apigateways.operator.kyma-project.io` CustomResourceDefinition (CRD) describes the kind and the format of data that APIGateway Controller uses to configure resources. See [APIGateway Custom Resource](./apigateway/README.md).

## APIRule Custom Resource

The `apirules.gateway.kyma-project.io` CRD describes the kind and the format of data the APIRule Controller uses to configure resources. See [APIRule Custom Resource](./apirule/README.md).

## RateLimit Custom Resource

The `ratelimits.gateway.kyma-project.io` CRD describes the kind and the format of data that RateLimit Controller uses to configure the request rate limits for applications. See [RateLimit Custom Resource](./ratelimit/04-00-ratelimit.md).
